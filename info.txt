создай несколько ts файлов, которые будут начинаться с цифры (чтобы идти по порядку), затем - название темы файла.
В каждом файле пропиши свой тип тайпскрипта (комментарием) и пример на этот тип.
То есть, для каждого типа будет свой файл. Для этого создашь папку с названием typescript-tutorial.

Пример структуры папки:

1-string.ts
2-integer.ts
...
..-object.ts
..-generic.ys
и т.д.

Пример содержимого файла 1-string.ts

// Тип string

const s: string = "abc"
